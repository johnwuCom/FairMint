# FairMint

## 參數說明

- `tokenAdd`: `"0x3e290e20d5c207523f53fa384ad87d69cc543f21"`  
  - 代幣合約地址，表示要交易的 **BEP-20 代幣**。

- `address`: `""`  
  - 用戶的 **錢包地址**，發送交易的來源地址。

- `privateKey`: `""`  
  - 對應 `address` 的 **私鑰**，用於 **簽署交易**。

- `BNB_Amount`: `0.01`  
  - 交易中涉及的 **BNB 數量**，例如 **兌換 0.01 BNB 為目標 Token**。
